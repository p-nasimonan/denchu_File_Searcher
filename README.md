# denchu_File_Searcher
 デンチュウさんのプログラムで変更されたところを見る（多分最初にあるPATHを変えればなんでも比較できるかも)

## 注意
著作権とかの都合上でんちゅー本体やRedmeは消しましたのでresult.txtと標準出力が異なる可能性があります。

## 動機
デンちゅーのプログラムが授業で毎回ダウンロードするのでなぜかと思った。

## 結果(雑に書いたから多分バグがなければ)
configしか変わらない！

## 実行方法
これを実行する
```
denchu_diff.py
```


ファイルに出力するのもあり
```
denchu_diff.py > result.txt
```
